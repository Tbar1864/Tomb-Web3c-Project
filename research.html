My Research Assignments

"For class research , I completed three class assignments. The first on the World Wide Web and understanding the historical and foundational technologies related to it. The second on analyzing the application of HTML/CSS in modern web design. The third is a Power Point comparison of two popular CSS frameworks. I also created a quiz page to test your knowledge of the topics covered in the assignments. A brief description of each assignment is provided below."

"Assignment 1: The World Wide Web - This assignment focuses on the application of HTML and CSS in modern web design. It explores the role of HTML as the backbone of web content and details the history and foundational technologies of the World Wide Web (WWW)."

"Assignment 2: HTML/CSS - The report analyzes the "Band Template" from w3schools.com, highlighting its clean, simple, and responsive design. Overall, the template is effective as a base but requires adjustments for better accessibility."

"Assignment 3: CSS Frameworks PowerPoint - The PowerPoint compares the CSS Frameworks Bootstrap and Bulma. It starts with a side-by-side comparison of the two frameworks, highlighting their similarities and differences. It continues into a deeper dive of some of the main features."
